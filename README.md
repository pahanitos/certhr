# CertHR AI Lab — landing site

Статический лендинг для новой концепции консалтинга: AI-инструменты для HR-процессов и управленческих HR-решений.

## Концепция

CertHR AI Lab помогает компаниям превращать HR-задачи и данные в объяснимые решения:

1. Компания формулирует HR-вопрос.
2. Передает данные: Excel, HRIS-выгрузки, опросы, политики, оргструктуру.
3. AI-модель анализирует ситуацию по экспертной методологии.
4. На выходе формируются рекомендации, сценарии, риски, отчеты и материалы для внедрения.

## Ключевые файлы

- `index.html` — контент и структура лендинга.
- `assets/styles.css` — дизайн-система и адаптивные стили.
- `assets/app.js` — переключение темы, мобильное меню, FAQ и scroll reveal.
- `downloads/free/` — бесплатные Excel-файлы для скачивания.

## Основные секции

- Hero: AI-инструменты для HR-решений.
- Почему это нужно: проблемы ручного HR-анализа.
- Как работает: задача → данные → AI-модель → решение.
- AI-инструменты: модули для compensation, salary review, бонусов, job architecture, retention, engagement, policies и org design.
- Форматы работы: AI-аудит, кастомный инструмент, HR AI Toolkit, интеграция и обучение.
- Флагман: AI Salary Review Copilot.
- Продуктовая линейка: AI Diagnostic, AI Tool Setup, HR AI System.
- Ресурсы: бесплатные Excel-файлы и платные AI-модули по запросу.
- FAQ и финальный CTA.
